# Express Node.js example by UlbiTV

Backend для начинающих на node.js express. RESTfull API, CRUD, HTTP

https://www.youtube.com/watch?v=tKM44vPHU0U:

Dev-сервер (с отслеживанием изменений кода - nodemon):
`npm run dev`

`npm run start`
